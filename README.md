# Aulas Python 🐍

Este repositório contém códigos e anotações referentes aos estudos da linguagem Python. Utilizo este espaço para consolidar meu aprendizado e manter um histórico prático da evolução nos fundamentos da programação.

## 📁 Estrutura do Repositório

Os materiais estão organizados de forma progressiva, divididos por aulas e temas:

  * **[Aula01 - Variaveis e Tipos de dados](https://www.google.com/search?q=./Aula01%2520-%2520Variaveis%2520e%2520Tipos%2520de%2520dados):** Conceitos iniciais, declaração de variáveis e tipos primitivos na linguagem.
  * **[Aula02 - Operadores](https://www.google.com/search?q=./Aula02%2520-%2520Operadores):** Utilização de operadores aritméticos, lógicos e relacionais/de comparação.
  * **[Aula03 - Condicionais](https://www.google.com/search?q=./Aula03%2520-%2520Condicionais):** Estruturas de controle de fluxo lógico utilizando `if`, `elif` e `else`.
  * **[Aula04 - Saidas de dados](https://www.google.com/search?q=./Aula04%2520-%2520Saidas%2520de%2520dados):** Diferentes métodos para formatação de strings e manipulação da função `print()`.
  * **[Aula05 - Lacos de rep](https://www.google.com/search?q=./Aula05%2520-%2520Lacos%2520de%2520rep):** Aplicação prática das estruturas de repetição iterativas (`for` e `while`).
  * **[Aula06 - Modulos](https://www.google.com/search?q=./Aula06%2520-%2520Modulos):** Introdução à importação, criação e uso de módulos para modularização do código.
  * **[Aula07 - Modulo random](https://www.google.com/search?q=./Aula07%2520-%2520Modulo%2520random):** Exemplos e testes utilizando a biblioteca padrão `random` para geração de dados aleatórios.
  * **[modulos e pacotes.txt](https://www.google.com/search?q=./modulos%2520e%2520pacotes.txt):** Anotações teóricas complementares sobre a estruturação de projetos em Python.

## 🛠️ Tecnologias e Ferramentas

  * **Python 3**
  * **Git e GitHub**
